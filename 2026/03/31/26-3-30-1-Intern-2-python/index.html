Python 爬虫基础

爬取网站数据的过程

  1. 发送请求：模仿浏览器点击。
  2. 获取响应：收到 HTML 源代码。
  3. 解析内容：从 HTML 中提取有效数据。
  4. 存储数据：将数据存进文件或数据库。

常用的爬虫库与功能

库 功能 优点 场景
requests 发送网络请求 简单易用 入门级爬虫
BeautifulSoup 解析 HTML 易读性好 静态网页解析
lxml 高性能解析 快速稳定 大规模数据提取
Scrapy 爬虫框架 模块化强 中大型项目
selenium 模拟浏览器操作 能执行 JavaScript 动态网页爬取
aiohttp 异步请求 并发高 海量网页采集
re 正则匹配 通用工具 提取特定文本内容

  1. 为什么说 selenium 能执行 JS 就能爬取动态网页？

    selenium 本身不能执行 JavaScript，也没有浏览器，而是通过 WebDriver 协议控制外部浏览器。在静态网页中，服务器发送的 HTML 代码即可表示网页所有内容；在动态网页中，HTML 代码还有 JavaScript 向 API 发送请求。只有将 JavaScript 代码在浏览器环境中执行后，才能完整爬取网页。

  2. 下载 BeautifulSoup 的命令为 pip install beautifulsoup4。

find_all() 方法

参数使用与 find() 相似，但是此方法返回一个列表，包含所有满足条件的元素。


CSS 选择器 select() 与 select_one()

select() 与 select_one() 通过 CSS 寻找元素。

实际开发中的最佳实践：

  • 简单查找（按标签、class、id）使用 find() / find_all()
  • 复杂的层级查找使用 select() / select_one()
  • 非常复杂的场景可以先用 select() 定位到大致区域，再用 find() 提取细节

文本内容提取 text() 与 get_text()

text() 直接返回元素所含文本，get_text() 可以去除首尾空白。

robots.txt

robots.txt 是表明允许机器访问的网站路径的文件，一般位于网站根目录下，即 www.example.com/robots.txt。例如 Bilibili 的 www.bilibili.com/robots.txt
