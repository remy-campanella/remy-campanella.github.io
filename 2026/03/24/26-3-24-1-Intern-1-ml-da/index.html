机器学习的思路

问题：寻找一条直线，“平衡”所有的点。

根据图片，给出一个方程
\[ y = wx + b \]

找到一组 \((w, b)\) 所有的点到直线的距离（原始数据与预测值的 \(y\) 值之差）的和最小（使用 for 循环遍历计算）。

绝对值距离也可由平方代替。
\[ l = \sum_{i = 1}^n(wx_i + b - y_i)^2 \]

这个过程就是机器学习的过程，将这个过程封装到一个方法中，这个方法就是机器学习所说的算法。

根据前面推断，不断画直线，找到 \((w, b)\)，使 \(l\) 最小。这就是最小二乘法。

最小二乘法

设由数据集
\[ D = {(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)} \]

设最小二乘法
\[ Q = \sum_{i = 1}^n(kx_i + b - y_i)^2 \]
求当 \(Q\) 最小时 \(k\) 和 \(b\) 的值。

对 \(Q(k, b)\) 求偏导数为 0 解极小值
\[ \frac{\partial Q}{\partial k} = \frac{\partial \sum_{i = 1}^n(kx_i + b - y_i)^2}{\partial k} = 0 \\ \frac{\partial Q}{\partial b} = \frac{\partial \sum_{i = 1}^n(kx_i + b - y_i)^2}{\partial b} = 0 \]

解得
\[ k = \frac{\sum_{i = 1}^n(x_1 - \bar{x})(y_i - \bar{y})}{\sum_{i = 1}^n(x_i - \bar{x})^2} \\ b = \bar{y} - k \bar{x} \]

机器学习的种类

算法对比

每种机器学习的底层都是用不同的算法来计算损失。

算法类别 类型 典型应用场景 评估指标
线性回归 (Linear Regression) 监督学习 (回归) 房价预测、销量预估、趋势分析 MSE, RMSE, MAE, R²
逻辑回归 (Logistic Regression) 监督学习 (分类) 用户流失预测、垃圾邮件识别、疾病诊断（二分类） Accuracy, Precision, Recall, F1-Score, AUC-ROC
决策树 (Decision Tree) 监督学习 (分类/回归) 信贷审批规则生成、可解释性要求高的决策场景 Accuracy, F1-Score, Gini Impurity
随机森林 (Random Forest) 监督学习 (分类/回归) 金融风控、特征重要性分析、抗过噪声数据 Accuracy, AUC-ROC, OOB Score
支持向量机 (SVM) 监督学习 (分类/回归) 小样本高维数据分类、文本分类、图像识别 Accuracy, F1-Score, Hinge Loss
K-近邻 (KNN) 监督学习 (分类/回归) 推荐系统（基于相似性）、简单分类任务 Accuracy, Precision, Recall
朴素贝叶斯 (Naive Bayes) 监督学习 (分类) 文本情感分析、新闻分类、实时性要求高的场景 Accuracy, Log-Loss, F1-Score
梯度提升树 (XGBoost/LightGBM) 监督学习 (分类/回归) Kaggle 竞赛、结构化数据建模、点击率预估 (CTR) AUC-ROC, Log-Loss, RMSE
K-Means 聚类 无监督学习 (聚类) 客户分群、市场细分、图像压缩 轮廓系数 (Silhouette), 惯性 (Inertia)
DBSCAN 无监督学习 (聚类) 异常检测、密度不均的聚类、任意形状簇发现 轮廓系数，可达性图
主成分分析 (PCA) 无监督学习 (降维) 数据可视化、特征去噪、降维加速后续模型 解释方差比，重构误差
神经网络/深度学习 (NN/DL) 监督/无监督 (复杂模式) 图像识别、NLP、语音识别、复杂非线性关系建模 Accuracy, Loss, Perplexity (NLP)

常见评估指标

回归任务指标

名称 解释
MSE 均方误差，对异常值敏感，越小越好。
RMSE 均方根误差，作用与 MSE 相同，量纲与目标变量相同，好解释。
\(R^2\) 决定系数，模型拟合优度，越接近 1 越好。

分类任务指标

名称 解释
Accuracy 准确率，适用于类别平衡。
Precision 查准率，预测为“正”的样本中实际为“正”的比例，“宁缺毋滥”。
Recall 查全率，实际为“正”的样本中被正确预测的比例，“宁错毋漏”。
F1-Score Precision 与 Recall 的调和平均数。
AUC-ROC 曲线下面积，模型区分正负样本的能力，适用于不平衡数据。

聚类任务指标

名称 解释
Sihoulette Score 轮廓系数，簇内紧密度与簇内分离度。
Inertia 惯性，簇内平方和，K-Means 中用于寻找最佳 K 值。

监督学习与无（非）监督学习

  1. 监督学习
    原始数据中具有明显的“问题—答案”结构。机器在原始数据中进行学习，然后预测新“问题”的“答案”。

  2. 无监督学习
    原始数据中只有单纯的数据，没有“答案”，机器需要从中发现数据的结构与特征，多用于聚类。

如何使用机器学习

根据数据情况使用机器学习算法。

使用原始数据进行机器学习训练是否适合？

原始数据有异常，不能直接用于机器学习，需要先对数据进行分析和处理。

  1. 不均衡：上采样、下采样
  2. 存在空值：使用最大、最小、均、中位等填补
  3. 存在异常值：归一化
