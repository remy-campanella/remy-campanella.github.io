AI in the World Today

In-class Quiz: Why Do We Call These Systems Intelligent?

There Is No Single Definition of AI

Working view for this course: AI is a scientific field that studies the mechanisms of intelligent behaviour and thinking-like problem solving, especially through the analysis and design of agents.

What Makes AI Different from Ordinary Software?

Agent as an Unifying Framework of AI

An agent is a system that perceives its environment and acts upon that environment in pursuit of some objective.

PEAS: A Simple Framework for Describing Agent Tasks

We describe agent tasks using PEAS framework:

Letter Meaning
Performance What counts as success?
Environment What world does the agent operate in?
Actuators How can the agent act?
Sensors What information can the agent recieve?
