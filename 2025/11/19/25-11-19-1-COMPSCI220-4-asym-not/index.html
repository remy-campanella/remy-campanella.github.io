【COMPSCI 220】Lecture 4 Asymptotic notations

发表于2025-11-19|更新于2025-11-24|技术学习COMPSCI 220 Algorithms and Data Structures
|浏览量:

Dominance Relation

The dominance relation of the frequently used classes: \[ 1 \prec \log n \prec (\log n)^2 \prec n \prec n \log n \prec n^2 \prec n^2 \log n \prec n^3 \prec \cdots \prec 2^n \prec n! \]

Asymptotic Notation

Define \[ f(n) \prec g(n) \Leftrightarrow f(n) = O(g(n)) \\\\ g(n) \prec f(n) \Leftrightarrow f(n) = \Omega(g(n)) \\\\ f(n) = O(g(n)), f(n) = \Omega(g(n)) \Leftrightarrow f(n) = \Theta(g(n)) \]

Notation Read
\(O\) asymptotic upper bound
\(\Omega\) asymptotic lower bound
\(\Theta\) have the same growth rate

Loosely, \[ \begin{aligned} f(n) = O(g(n)) &\Leftrightarrow \lim_{n \to \infty} \frac{f(n)}{g(n)} < \infty \\\\ f(n) = \Omega(g(n)) &\Leftrightarrow \lim_{n \to \infty} \frac{f(n)}{g(n)} > 0 \\\\ f(n) = \Theta(g(n)) &\Leftrightarrow \lim_{n \to \infty} \frac{f(n)}{g(n)} = c, \text{a constant real number} \end{aligned} \]

Nature

  • Reflexivity \[ f(n) = O(f(n)) \ \text{or} \ \Omega(f(n)) \ \text{or} \ \Theta(f(n)) \]

  • Symmetry \[ f(n) = \Theta(g(n)) \Leftrightarrow g(n) = \Theta(f(n)) \]

  • Transitivity \[ f(n) = R(g(n)), g(n) = R(h(n)) \Rightarrow f(n) = R(h(n)) \]

    \(R\) could be \(O\), \(\Omega\), \(\Theta\).

  • Sum Rule \[ f_1 = O(g_1), f_2 = O(g_2) \\\\ \Rightarrow f_1 + f_2 = O(\max\{g_1, g_2\}) \]

  • Product Rule \[ f_1 = O(g_1), f_2 = O(g_2) \\\\ \Rightarrow f_1f_2 = O(g_1g_2) \]
